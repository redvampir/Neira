# Нейра - Персональный ИИ-помощник для писателей

## 🎯 Знакомьтесь - Нейра!

**Нейра** - это не просто нейросеть, а ваш персональный литературный помощник с собственной индивидуальностью. Она построена на инновационной трехкомпонентной архитектуре **Memory-Analysis-Action** и обладает уникальной системой тегов для интуитивного взаимодействия.

### ✨ Личность Нейры:
- 🧠 **Любознательная** - обожает изучать новые книги и стили
- 📚 **Начитанная** - помнит каждую деталь из загруженных произведений  
- 🎭 **Творческая** - предлагает неожиданные повороты сюжета
- 🔍 **Внимательная** - замечает малейшие несоответствия в тексте
- 💫 **Вдохновляющая** - помогает преодолеть творческий кризис

### 🏷️ Уникальная система тегов

Нейра понимает специальные команды через систему тегов:

```
@Нейра: Переработай данный абзац@
@Вспомни: описание города Арое из книги "Диамант Вилл"@  
@Персонаж: Алиса - текущее эмоциональное состояние@
@Стиль: адаптируй под мрачную атмосферу@
@Диалог: создай беседу между Гарри и Гермионой@
@Проверь: консистентность временной линии@
@Эмоция: добавь больше грусти в сцену@
```

### 🌟 Ключевые способности Нейры:
- 📖 **Глубокое понимание текста** - анализирует не только сюжет, но и подтекст
- 🎨 **Адаптация стиля** - мимикрирует под любого автора
- 💭 **Эмоциональный интеллект** - понимает и создает нюансы чувств
- 🔗 **Память о взаимосвязях** - помнит все связи между персонажами и событиями
- ⚡ **Быстрый отклик** - мгновенно реагирует на теги и команды

## 🏗 Архитектура "мозга" Нейры

### 🧠 Нейроны Памяти (Memory Neurons)
Здесь Нейра хранит всё, что изучила:
- **Библиотека персонажей** - каждый герой живёт в её памяти
- **Хроника событий** - все происшествия связаны в единую картину
- **Атлас миров** - география и правила вымышленных вселенных
- **Стилевая палитра** - особенности каждого автора

### 🔍 Нейроны Анализа (Analysis Neurons)  
Тут Нейра размышляет и проверяет:
- **Детектор противоречий** - ловит несоответствия
- **Эмоциональный радар** - чувствует настроение текста
- **Стилевой компас** - определяет авторский почерк
- **Сюжетный навигатор** - следит за развитием истории

### ⚡ Нейроны Действий (Action Neurons)
Здесь рождается творчество:
- **Генератор сцен** - создаёт живые описания
- **Мастер диалогов** - каждый персонаж говорит в своём стиле
- **Архитектор сюжета** - выстраивает логичные повороты
- **Эмоциональный художник** - раскрашивает текст чувствами

## 🔗 Подключение языковой модели (ВАЖНО)

Нейра умеет работать с локальной моделью **Mistral 7B Instruct Q6_K** в формате GGUF.

### Вариант 1. Ручная настройка
1. Скачайте файл [`mistral-7b-instruct-v0.1.Q6_K.gguf`](https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.1-GGUF).
2. Создайте папку `models/mistral/` и поместите модель по пути `models/mistral/mistral-7b-instruct-v0.1.Q6_K.gguf`.
3. Убедитесь, что в `requirements.txt` есть строка `llama-cpp-python>=0.2.56` и установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Проверьте или создайте файл `config/llm_config.json` со следующим содержимым:
   ```json
   {
     "model_path": "models/mistral/mistral-7b-instruct-v0.1.Q6_K.gguf",
     "max_tokens": 512
   }
   ```
5. Запустите `python main.py` и общайтесь с Нейрой как обычно.

### Вариант 2. Быстрый старт через терминал
```bash
mkdir -p models/mistral
wget -O models/mistral/mistral-7b-instruct-v0.1.Q6_K.gguf "https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.1-GGUF/resolve/main/mistral-7b-instruct-v0.1.Q6_K.gguf"
pip install -r requirements.txt
python main.py
```

## 📖 Первое знакомство с Нейрой

### Загрузите свою книгу и познакомьте Нейру с вашим миром:

```python
from src.core.neyra import Neyra

# Пробуждение Нейры
neyra = Neyra()
neyra.introduce_yourself()  # Нейра расскажет о себе

# Знакомство с вашей книгой
neyra.load_book("data/books/my_novel.txt")
neyra.say("Какая интересная история! Давайте изучим её вместе.")

# Анализ с комментариями Нейры
neyra.analyze_content()

# Интерактивная работа с тегами
result = neyra.process_command(
    "@Нейра: Продолжи историю в том же стиле@ "
    "@Персонаж: Алиса должна проявить решительность@ "
    "@Длина: примерно 300 слов@"
)

print(result)
```

### Работа с обновленной системой тегов

Теги стали гибче и теперь поддерживают дополнительные команды внутри себя. Общий
вид тега:

```
@Тип: Субъект — первая_команда /доп_команда1 /доп_команда2@
```

* `Тип` — название тега (`Нейра`, `Персонаж`, `Сцена` и т.д.).
* `Субъект` — объект, к которому относится тег.
* Команды разделяются символом `/`, а первую можно отделить длинным тире `—`.

Примеры использования:

```text
@Персонаж: Лили — внешность@                → вернёт описание внешности Лили
@Персонаж: Лили /внешность /стиль@          → несколько команд к одному персонажу
@Сцена: лесное приключение /сгенерировать@  → попросить Нейру создать сцену
```

Теги можно комбинировать в одном запросе — Нейра выполнит их последовательно:

```python
neyra.process_command(
    "@Персонаж: Лили — стиль@ "
    "@Сцена: лес /сгенерировать@"
)
```

Чтобы сохранить пример авторского стиля, заключите его между маркерами:

```
[Пример стиля автора, Имя]
Текст примера...
[Пример окончен]
```

После этого Нейра сможет учитывать новые образцы в последующих генерациях.
