# Руководство пользователя

Приложение организовано вокруг режимов. Начните с учебного режима, чтобы
изучить интерфейс. После знакомства вы можете переключиться на другие режимы,
такие как менеджер ресурсов, чтобы импортировать и искать собственные данные.

Плагины могут добавлять новые функции. Включите их в файле конфигурации, и они
будут автоматически запускаться во время генерации. Подробнее о написании
плагинов см. в [руководстве разработчика](../developer/README.ru.md).

## Мини-чат

В редактор встроено лёгкое окно чата для быстрых вопросов.
Нажмите `Ctrl+Shift+M`, чтобы открыть мини-чат в любой момент.
Также можно кликнуть правой кнопкой в редакторе и выбрать пункт
**«Мини-чат…»** из контекстного меню.
