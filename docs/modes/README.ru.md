# Режимы

Neira предоставляет отдельные режимы, которые разделяют общий интерфейс. Каждый
режим можно запустить и остановить, и он может передать управление другому
режиму после завершения своей задачи.

## Взаимосвязи

- **Учебный режим** проводит новых пользователей через начальную настройку,
  после чего передает управление другим режимам.
- **Менеджер ресурсов** работает с локальными ресурсами. Данные, подготовленные
  здесь, доступны любому последующему режиму.

Все режимы имеют доступ к общим службам, таким как шина событий и система
конфигурации, что обеспечивает плавные переходы между ними.
