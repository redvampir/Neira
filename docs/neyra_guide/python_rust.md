# Роли Python и Rust

Нейра использует два языка программирования:

- **Python** отвечает за высокоуровневую логику, оркестрацию модулей и быструю
  разработку. В нём реализованы плагины, взаимодействие с пользователем и
  эксперименты.
- **Rust** применяется для вычислительно затратных компонентов. На нём
  реализованы структуры данных `MemoryIndex`, обработка тегов и другие
  производительные части.

## Рекомендации по генерации кода

1. Прототипируйте новые возможности на Python. После стабилизации
   выносите горячие участки в Rust.
2. Межъязыковой интерфейс оформляйте через `maturin` и библиотеку PyO3.
3. Передавайте через границу простые типы данных и избегайте shared state.
4. Для Python-обёрток используйте аннотации типов и понятные docstring.
5. После изменения Rust-кода выполняйте `maturin develop`, чтобы обновить
   расширение.
