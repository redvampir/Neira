# Обзор документации

Эта папка содержит руководства по режимам Neira, информацию для разработчиков и
инструкции для пользователей. Каждый раздел опирается на базовые концепции,
такие как система режимов и архитектура плагинов, чтобы сохранять модульность и
расширяемость проекта.

- [Режимы](modes/README.ru.md) – как взаимодействуют рабочие режимы.
- [Руководство разработчика](developer/README.ru.md) – стандарты интерфейса и
  структура плагинов.
- [Руководство пользователя](user-guide/README.ru.md) – инструкции по
  использованию Neira.
